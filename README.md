# binary-search-tree

Binary Search Tree using JavaScript

# Tools:

- JavaScript
- Jest
- Babel (for Jest ES6+ compatibility)
- Prettier (VSCode ext.)

# Methods:

- `includes(value)`
- `insert(value)`
- `deleteItem(value)`
- `levelOrderForEach(callback)`
- `levelOrderForEachLoop(callback)`
- `inOrderForEach(callback)`
- `preOrderForEach(callback)`
- `height(value)`
- `depth(value)`
- `isBalance()`
- `rebalance()`

# What I learned:

- Implementing BST.
- Implementing Queue.
- Better understanding of recursion.

# Requirements:

- git
- NodeJS

# How to clone:

1. Clone using this command: git clone `https://github.com/SuspiciousApplePie/binary-search-tree.git.`
2. Install Jest: npm install --save-dev jest.
3. Install Babel: npm install --save-dev @babel/preset-env.
4. Done.
